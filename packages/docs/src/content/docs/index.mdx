## Let's get started

<Steps>

1. Install your preferred JavaScript Package Manager

2. Run the Giolt CLI

   <Tabs syncKey="package-manager">

   <TabItem label="npm" icon="npm">
  
   ```sh
   npx jsr @giolt/cli
   ```
  
   </TabItem>
  
   <TabItem label="bun" icon="bun">
  
   ```sh
   bunx jsr @giolt/cli
   ```
  
   </TabItem>
   
   <TabItem label="deno" icon="deno">
  
   ```sh
   deno run jsr:@giolt/cli
   ```
  
   </TabItem>
  
   </Tabs>
</Steps>

:::note
The CLI tool is still in development
:::
